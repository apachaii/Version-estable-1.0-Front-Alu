¿Como hablo con Ada?

Para poder hablar con Ada deberán tener en cuenta que funcionará como cualquier chat que ya hallas utilizado antes, simplemente da clic dentro del rectangulo que indica la imagen y podras escribir.

¿Como selecciono una asignatura?

Para seleccionar una asginatura deberas en a la barra de menu que se encuentra en la parte superior y pinchar la opcion "Asignatura", esto te mostrara las asignaturas a las que Ada podra darte su ayuda, solo deberas pinchar la que necesites para que te redireccione al chat de respectiva asignatura.

¿Como enviar una sugerencia?

Dentro del menu se encuentra la opcion "Sugerencias" para que nos envies tu opinion respecto al funcionamiento de Ada, para realizar este proceso sera ten sencillo como pinchar en las estrellas para calificanor y luego escribir un comentario sobre Ada, pero recuerda quiereambos campos son necesarios para poder dar en el boton de enviar.

¿De donde se obtiene la información?

Recuerda que Ada siempre te entregara informacion actualizada y totalmente confiable para demostrar esto existe una seccion de "Referencias" dentro de nuestro menu que al ingresar te mostrara de donde se obtiene la información que Ada te esta entregando
